# python_crawler
This is a repository about some python crawler demos.

- 1-bing_com_search_result.py
	-- Using bs4 to extract data from bing.com search results.
	
- 4-douban_movies_top250_regex.py
	-- Using regex to extract data from douban movies top250

- 5-douban_movies_top250_BeautifulSoup.py
	-- Using BeautifulSoup to extract data from douban movies top250
	
- 5-douban_movies_top250_xpath.py
	-- Using XPath to extract data from douban movies top250
	
- 7-taobao_products_selenium.py
	-- Using Selenium to extract data from taobao search results.
